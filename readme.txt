Indian Premier League
This is a Command line excercise. Use cli-table package.

Questions
Display Year,City, Team1 Name, Team2 Name,Result, Winner, Win By Runs, Win By Wickets, Player Of the Match data in CLI table in descending order of Year.

Pass Year as argument and sort data by Date and display Date, City, Team1 Name, Team2 Name, Result, Winner, Win By Runs, Win By Wickets, Player Of the Match data in CLI table.

Pass Year as argument and create Points table of that year and display Team Name, Matches, Won, Lost, Tied, No Result, Points data in CLI table.

Consider following table for point calculation

Points	Result
2	Won
1	No Result
1	Tie
0	Loss
